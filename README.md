# Mineradio-ShinaYuu-Ver-
Mineradio là trình phát nhạc tương tác dành cho máy tính để bàn Windows, kết hợp Discord player  , phát nhạc tìm kiếm, hiển thị lời bài hát ( Spotify ), hiệu ứng hình ảnh hạt và kệ danh sách phát 3D vào một không gian âm nhạc riêng tư, mang lại cảm giác như một buổi biểu diễn trực tiếp.
