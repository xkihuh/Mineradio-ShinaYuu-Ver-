# ShinaYuu Music 1.1.1

Đây là mốc nền tảng trước các thay đổi ngày 2026-07-12.

- Ứng dụng desktop Electron với giao diện ShinaYuu Music.
- Phát Spotify trực tiếp bằng Web Playback SDK và phát YouTube bằng yt-dlp.
- Spotify OAuth, hồ sơ người dùng, tìm kiếm, playlist và hàng chờ.
- Visual Effects, Desktop Lyrics, fullscreen và hiệu ứng nhịp thời gian thực.
- Discord Profile Card và Rich Presence cục bộ.
- Build Windows bằng Electron Builder + NSIS.
